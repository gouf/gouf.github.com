Octpress のテーマを変更する

デフォルトのテーマがあんまり好きじゃないので、よさそなのに変える。

$ cd octopress $ git clone git://github.com/tommy351/Octopress-Theme-Slash.git .themes/slash $ rake install['slash'] $ rake generate
